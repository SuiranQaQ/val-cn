VAL CN 便携版（免安装）
（本文件请用记事本打开 README.txt）

【重要：正常 UI 在浏览器里】
本工具是完整网页界面（和 valcn 一样），不是黑窗口程序。
双击 VAL-CN.vbs → 自动打开浏览器 → 那就是正常 UI（查战绩、统计、长图等）。
bat 只是另一种启动方式，界面完全相同。

【游戏电脑怎么用】
1. 解压 VAL-CN-portable.zip
2. 不确定 Token？先双击 check.bat 看日志
3. 双击 VAL-CN-debug.bat（推荐）— 先检查再启动，本窗口有日志
4. 或双击 VAL-CN.vbs（无黑窗口，静默启动）打开战绩站
5. 打游戏时：先开瓦罗兰特 → 双击 VAL-CN-live.vbs 打开对局认人

【停止服务】
双击 stop.bat 关闭后台服务

【无需安装】
已内置 Node.js，不需要 npm 环境。

【Token】
开着游戏会自动读客户端会话；也可编辑 .env.local 填入抓包 Token。

【注意】
- 仅本机 Windows 使用
- 对局认人必须瓦罗兰特在本机运行

【与反作弊】
不读游戏内存、不注入进程，与外挂无关；仅供战绩查询参考。
